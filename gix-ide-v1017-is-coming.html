Gix-IDE v1.0.17 is coming

The new version of Gix-IDE is coming. This is basically a "refresh" release that incorporates all the latest developments in GixSQL. There are no changes to the IDE or the debugger.

You can find a summary of the new features and fixes in GixSQL here.

Packages have already been built, I just need to test them (a few modifications have been made to the packaging system that needs to be tested) but they will probably be available during the next week.

P.S. there is no published v1.0.16 version of Gix-IDE, I jumped directly from 1.0.15 to 1.0.17 to keep the versioning of Gix-IDE and GixSQL aligned. Gix-IDE v1.0.16 only exists as an internal version.

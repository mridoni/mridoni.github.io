GixSQL 1.0.16 is coming

Gix-SQL 1.0.16 is (almost) ready. This release (that will also be released as soon as possible into Gix-IDE) will mainly add:

  1. Much improved logging
  2. EXEC SQL WHENEVER support
  3. New format for "connection strings"
  4. The MySQL driver now links against (and ships with) the MariaDB client libraries
  5. Several binary dependencies are now installed with vcpkg (on Windows)

Due to 4) and 5) I am overhauling the build/test system, so the binary packages are taking a bit longer than expected, but they should be ready in or two more weeks.

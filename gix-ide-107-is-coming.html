Gix-IDE 1.0.7 is coming

Gix-IDE 1.0.7 is coming

The new version of Gix-IDE will be published around mid-January. It will ship with several fixes in the ESQL preprocessor, both the version integrated in Gix-IDE and the standalone one.

Separate install packages for GixSQL will also be available (for Windows and Linux, as usual) and it will be possibile to build GixSQL only instead of the whole IDE.

This version will also integrate assorted fixes for issues like debugging on Windows/x86 and startup errors.

Check this page, GnuCOBOL's contribution GnuCOBOL - Contributions or GitHub's Gix-IDE Releases page for an update

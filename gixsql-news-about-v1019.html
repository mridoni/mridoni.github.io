GixSQL: news about v1.0.19

The internal branch v1.0.19dev has been merged into main and it will be the base for the upcoming version v1.0.19. It implements, as announced, support for autocommit and updatable cursors. It includes, obviously, some fixes that have been applied to the main branch in past few weeks.

This merge will also allow to integrate fixes for other issues more easily. The final release of v1.0.19 should be out in a week, two at most, then - barring the need for hotfixes - work will resume on Gix-IDE with two main targets: complete functionality for the standalone debugger and an enhanced error/output window.
